**Task 2 - Future Interns Cyber Security Internship**

- Email Header Analyzers
- Browser DevTools

3. Identified common phishing indicators
4. Classified risk levels (Low/Medium/High/Critical)
5. Created professional awareness report

- Prevention guidelines for users
